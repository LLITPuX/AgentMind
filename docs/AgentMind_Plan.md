## Покроковий план розробки платформи "AgentMind"

- **Версія**: 1.0  
- **Остання зміна**: 20.10.2025

### 1. Загальна концепція та мета

Розробляємо комплексну платформу "AgentMind" для керування пам'яттю самонавчальних ШІ-агентів. Платформа повністю контейнеризована (Docker) і складається з бекенду на Python та фронтенду на Next.js.

- **Основні цілі**:
  - **Надійна система пам'яті**: короткострокова (STM) і довгострокова (LTM) на базі FalkorDB.
  - **Просунута логіка**: оркестрація процесів мислення за допомогою LangGraph (консолідація пам'яті, пошук).
  - **Бітемпоральність**: облік актуальності фактів у часі (valid_from/valid_until та tx_time).
  - **Гібридний пошук**: поєднання графового та векторного пошуку для релевантного пригадування.
  - **Чистота даних**: дедуплікація для уникнення дублювання сутностей у графі знань.
  - **Візуальний інтерфейс**: UI для моніторингу, налагодження та інтерактивної взаємодії з пам'яттю.

- **Методологія**: Ітеративна розробка, керована тестами (TDD). Кожен новий функціонал спочатку покривається тестами, а потім реалізується.
